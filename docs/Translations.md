# Translations

```javascript
module.exports.adminpanel = {
    translation: {
        locales: ['en', 'ru', 'de', 'ua'],
        path: 'config/locales/adminpanel',
        defaultLocale: 'en'
    },
}
```

- locales - list of locales
- path - relative path to translations from project directory
- defaultLocale - default locale, locales[0] if not defined
